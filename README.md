# Optic TV Pro

A premium, native Android TV application for high-quality IPTV streaming.

## Features
- **Modern UI**: Dark slate and gold themed glassmorphism design.
- **TV Optimized**: Built with Jetpack Compose for TV for perfect remote control navigation.
- **Code-based Login**: Activation code system synchronized with Firebase.
- **High Stability Player**: Powered by Media3 ExoPlayer.

## Project Structure
- `app/`: Native Android application module.
- `gradle/libs.versions.toml`: Version catalog for dependency management.
- `.github/workflows/`: CI/CD actions for automated builds.

## How to Build
1. Clone the repository.
2. Open in Android Studio.
3. Sync Gradle and build the `app` module.
